# Repository layout

**LorBeePlugin** is this repository: a **Docker Compose** stack (Zigbee2MQTT, Mosquitto, Node-RED, optional LoRaWAN node) you install on an existing **DietPi** or other Debian-based ARM system with Docker.

**LorBeeOS** is a separate effort: a **ready-to-flash** image for Raspberry Pi Zero 2W that bundles this stack with a curated filesystem layout. When LorBeeOS ships, its docs will describe image-specific paths; this repo stays the reference for the Compose project and DIY installs.

Install and configuration entrypoints: **README.md**, **`docs/configuration-map.md`**.
